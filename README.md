# 🛠️ Reparaciones MP - Cotizador



Una página web sencilla y moderna diseñada para que los clientes puedan solicitar una cotización de reparación de dispositivos directamente a WhatsApp.



\# 📸 Demo en vivo

https://reparaciones-mp.netlify.app/



\# ✨ Características



\- 📱 \*\*Diseño Glassmorphism\*\*: Estilo visual moderno con transparencias y desenfoque.

\- 🚀 \*\*Sin detección automática de dispositivo\*\*: Envía los datos tal cual el usuario los escribe para evitar errores.

\- ✅ \*\*Validación de Formularios\*\*: Limpia automáticamente el texto en los campos para evitar caracteres raros.

\- 📲 \*\*Envío directo a WhatsApp\*\*: Lleva a los clientes a una conversación de chat inmediatamente.



\# 🛠️ Tecnologías utilizadas



\- HTML5 Estructurado

\- CSS3 (Estilos nativos y animaciones)

\- JavaScript Vanilla (Lógica del formulario)

\- Tailwind CSS (Para el estilizado rápido y responsivo)



\# 🖥️ Cómo ejecutar este proyecto



Simplemente abre el archivo `index.html` en tu navegador web.



1\.  Descarga este repositorio a tu computadora.

2\.  Haz doble clic en `index.html`.



\# 📲 Configuración del número de WhatsApp



Para cambiar el número al que llegan las solicitudes, edita la línea `const TARGET\_PHONE` dentro de la etiqueta `<script>` del archivo `index.html`.
